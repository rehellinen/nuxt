跳蚤市场
==========
API、CMS、公众号

### 技术点介绍：
1. SSR：Nuxt.js + Koa2
2. API：graphQL + RESTful
3. ORM：bookshelf.js
4. MVC分层
5. async函数
6. 完整的Token令牌体系
7. 函数式编程
8. 中间件与AOP
9. decorator优化路由、Token以及数据校验策略
10. 用了少量的函数式编程
